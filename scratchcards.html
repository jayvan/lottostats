Scratchcards

Scratchcards offer fun games and instant wins, but it pays to play smart. When a new scratchcard game is created there are a set number of winning scratchcards, as time goes on these winners are sold. You don't want to buy a scratchcard if all the top prizes are gone, but lotto retailers will keep trying to sell them! Each day the remaining prizes are updated here on Lottery Galaxy to show you the most up to date odds for each scratchcard. All of the scratchcards are analysed using statistics to figure out how much money you can expect to win. On this page you can see the scratchcards with the biggest top prizes, the best returns, and the best chances of winning a prize. Use this information to see which scratchards are the best for you each day. For an introduction to what each of these stats means check out the Statistics Guide. Good luck!
